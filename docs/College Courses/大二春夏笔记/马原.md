        - 炫耀知识和炫耀财产本质相同？没有改变世界的目标（信仰）会导致心理的空虚，生活缺乏激情和动力。
        - 马克思主义者 - 信仰 = 没有信仰的唯物主义者
        - 将个人信仰和国家信仰结合，在个人奋斗的同时造福社会。
