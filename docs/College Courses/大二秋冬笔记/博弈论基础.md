## 2 术语和基本假设

1. 术语
    - action
    - strategy：行动前准备好的一套完整的行动方案
        - 不可观察性：已知行动不可反推策略
        - e.g. 应急预案：
            - 信息分类：最好分 5 个危险等级，有哪些可能的危险（措施的前提条件）
            - 责任到人
            - 具体措施
            - 时效性
        - e.g. 三十六计是行动，根据信息选择计策是策略
        - 中国的立法精神：公民有退让的义务 -> 只有在危及生命的情况下反击才是正当防卫，否则跑 + 报警是最佳策略
        - 在不清楚对方的反应策略之前，避免发生正面冲突是最佳策略
    - 均衡：所有参与者最优策略的组合
        - 事先做好约定，不存在外部强制力的情况下，每个参与者都有积极性遵守约定 -> 纳什均衡
        - 现状的暴政：改革需要理由，不改革不需要理由
2. 基本假设
    - 认知理性：判断自身利益
        - e.g. 套餐是否真的便宜？满足用户对于自身利益的判断，让人感觉赚到但不是真的赚到：电子版 50，印刷版 120，印刷+电子版 125 / 电子版 50，印刷+电子版 120
        - e.g. 峰终体验：满足用户对于自身利益的判断
        - e.g. 利用中庸之道 -> 折中的比两端的更好，利用高价产品抬高品牌整体评价，并使主要产品价格合理化 -> 主动创造两利相权取其重的场景，让消费者在自己给定的选项中选择
    - 行为理性：追求自身利益最大化
        - e.g. 有报销才买保险
        - 完全理性的选择未必是最好的选择
            - e.g. 蜈蚣博弈：展现出更看重对方利益，传递合作信号，让对方更愿意合作。做一个愿意吃亏的人，找一个更愿意吃亏的人做朋友。

## 5 懦夫博弈

实验1：双方各发一个 1-20 元的红包，发的少的人再发 10 块钱。

竞争博弈 -- 换位思考：知道对方是新手还是老手会更容易赢得博弈。

胜者容易高估实力的作用，败者容易高估运气的作用。

定义：

|    | 进 | 退 |
| ------- | ------ | -------- |
| 进 | -5, -5 | 10, 0 |
| 退 | 0, 10 | 0, 0 |

变式：

|    | 进 | 退 |
| ------- | ------ | -------- |
| 进 | -5, -5 | 6, 4 |
| 退 | 4, 6 | 0, 0 |

会削弱进的概率 -- 奖励需要适度，过度奖励会导致投机者的出现。

实验2：100 元拍卖，出价最高的人获得 100 元，出价之后不返还。

赢者通吃的行业容易出现过度竞争。合适的策略 -- 评估可能的损失，投入适量成本，达到平均收益，避免参与竞争。（物质的牺牲！精神的胜利！x）

不返还增加了门槛，减少了竞争人数，反而更有赢的机会。

过度竞争容易忘记初心。e.g. 运动员服用违禁药，竞技体育的伤病。如果有一种药吃了会获得所有比赛的冠军，但是五年之后就会死，有 50% 以上的运动员愿意吃。

过度竞争会损失未来竞争力。

## 7 混合策略

- 纯策略：一个情景只有一个决策
- 混合策略：一个情景，以某种概率分布随机选择决策

1. 举例：猜钱博弈

|    | 出 100 | 出 10 |
| ------- | ------ | -------- |
| 猜 100 | 100, -100 | -X, X |
| 猜 10 | -X, X | 10, -10 |

计算期望：$X\leq \sqrt{AB}$ 左右期望开始为正

2. 举例：剪刀石头布，预先说明自己会出什么

小得失相信人，大得失怀疑人。

3. 偷懒博弈 -> 混合策略的计算

V 是雇员的贡献，即雇员为雇主创造的价值； W 是雇员的工资， H 是雇员的付出。 C 是雇主检查的成本。同时假定 H < W < V ， W > C。

|    | 偷懒 | 不偷懒 |
| ------- | ------ | -------- |
| 检查 | -C, 0 | V-W-C, W-H |
| 不检查 | -W, W | V-W, W-H |

假设雇主检查的概率为 $\alpha$，雇员偷懒的概率为 $\beta$

雇主的期望收益为： $$\pi_1 = (-C)\alpha\beta+(-W)(1-\alpha)\beta+(V-W-C)\alpha(1-\beta)+(V-W)(1-\alpha)(1-\beta)\\ \frac{\partial\pi_1}{\partial\alpha}=W\beta-C$$

所以当 $\beta<\frac{C}{W}$ 时雇主倾向于不检查，但是雇员的工作强度增大；当 $\beta<\frac{C}{W}$ 时雇主倾向于检查，但是雇员的工作强度减小。$\beta=\frac{C}{W}$ 时达到均衡。

雇员的期望收益为： $$\pi_2 = W(1-\alpha)\beta+(W-H)\alpha(1-\beta)+(W-H)(1-\alpha)(1-\beta)\\ \frac{\partial\pi_2}{\partial\beta}=H-W\alpha$$

所以当 $\alpha<\frac{H}{W}$ 时雇员倾向于偷懒，但是雇主检查的成本降低；当 $\alpha<\frac{H}{W}$ 时雇员倾向于不偷懒，但是雇主检查的成本增加。$\alpha=\frac{H}{W}$ 时达到均衡。

延伸结论：在均衡条件下，$$\pi_1 = V-W-V\frac{C}{W}\\ \frac{\partial\pi_1}{\partial W}=\frac{VC}{W^2}-1$$

所以在均衡条件下，雇主给雇员的工资为 $\sqrt{VC}$ 自身收益最大。C 越大，雇员分到的剩余价值越多。选择容易偷懒的职业并且不偷懒，更容易成功。

## 9 制度建设

博弈论第一原理：游戏规则决定游戏结果

现象 - 行为 - 制度：
- 不能用现象解释现象，楼塌了不是因为地震，而是因为楼造的不好，能反应出房地产监管制度不完善
- 制度就是“如果...那么...”
- 要改变的是制度而不是人心（不具有建设性意义），不要试图改变利己心而是利用利己心：离岸价和到岸价
- 外因是条件（当下无法通过行为改变，以后可能可以），内因是根据（当下可以通过行为改变）：从解决问题的角度思考问题，决定未来的努力方向 -> 只为成功找方法，不为失败找借口 -> 在沟通中，针对解决方法提问而不是针对错误提问
