# 本科毕业出路经验分享

## 信息来源：宣兴琦



1. 保研
    - 推免 & 接收：保研了也不是一劳永逸了
    - 校内：成绩很重要，选导师还是按照绩点排名
    - 校外：绩点参考价值较小，竞赛和科研可能会影响较大，会有笔试和面试
2. 考研
    - 自律？能保研就保研，考研太考研人了
3. 出国
    - 推荐：早点跟导师搞好关系，找国内外的大牛写推荐会很有用
    - 语言：提前开始准备，考托福雅思



总结：准备保研就先把绩点搞上去，准备出国就一直保持语言的学习。
