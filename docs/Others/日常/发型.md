# 发型打理教程

## Abstract

1. 造型：
    - 发泥 / 发蜡：干发打理
    - 发油：湿发打理
        - 油基：难洗
        - 水基 :heavy_check_mark:
        - 凝胶基
2. 定型：发胶（=定型喷雾）
    - 水胶：需要时间定型
    - 干胶 :heavy_check_mark:
3. 保养：
    - 护发素：干枯分叉用
    - 发膜
    - 护发精油
    - 顺序：发膜 - 护发素 - 精油


## 发泥打理教程

0. 洗头吹干
1. 取指甲盖大小
2. 均匀涂在手上
3. 全头往后抓
4. 头顶和后脑勺打散
5. 整理刘海
6. 发胶定型
