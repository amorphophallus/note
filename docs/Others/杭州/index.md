### 上天竺、法喜寺、灵隐寺

### 良渚

1. 良渚古城遗址公园（预计时间一天）
   - 鹿苑
   - 欣赏风景为主
2. 良渚博物馆（预计时间 2 到 3 小时）

tips：
1. 良渚博物馆和古城遗址公园距离 5km，需要打车或者坐公交车
2. 古城遗址公园很大， *步行一圈要 2h+* ，建议坐观光车，留半天到一天的时间慢慢走
3. 下午五点就闭园了，但是还可以在公园里继续玩，只是没有观光车接送了。勇一点可以逛到很晚，甚至可以看落日，然后找就近的门口出：
   - 南门
   - 东门
   - 西北门
4. 20r 的观光车票比其他多人自行车或者电动车性价比高，不如和门票一起买了
